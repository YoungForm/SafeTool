手动测试用例清单（Safetool 前端）

概览
- 预览地址： http://localhost:4173/
- 浏览器：桌面版 Chrome/Edge（启用默认下载目录）
- 测试账号：user / pass（示例）

1. 登录与导航
- 前置：预览服务已启动
- 步骤：输入用户名与密码；点击登录
- 期望：显示导航按钮 合规自检、IEC 62061、验证矩阵、证据库、互通、系统配置

2. 登录失败
- 前置：预览服务已启动
- 步骤：输入错误凭据；点击登录
- 期望：显示错误提示；导航按钮不可用或不显示

3. 键盘触发登录
- 前置：预览服务已启动
- 步骤：聚焦登录按钮；按 Enter
- 期望：登录成功并显示导航

4. 合规自检与报告生成
- 前置：已登录
- 步骤：进入合规自检页面；点击执行自检；点击生成报告
- 期望：显示自检结果摘要；报告预览 iframe 可见

5. IEC62061 评估与报告
- 前置：已登录
- 步骤：进入 IEC 62061；点击执行评估；点击预览报告
- 期望：显示 SIL 与 PFHd 等指标；报告预览 iframe 可见

6. 验证矩阵加载
- 前置：已登录
- 步骤：进入验证矩阵；输入项目ID demo；点击加载
- 期望：显示 ISO13849-1、IEC62061 条目与结果

7. 矩阵增强（索引与检查）
- 前置：已登录
- 步骤：进入矩阵增强；点击索引；点击检查
- 期望：条款索引与检查结果显示，提示清晰

8. 矩阵同步操作
- 前置：已登录
- 步骤：点击同步矩阵
- 期望：显示同步完成提示或结果摘要

9. 证据库加载
- 前置：已登录
- 步骤：进入证据库；点击加载
- 期望：显示证据列表（如 证据1）与状态元数据

10. 证据校验（单个与批量）
- 前置：已登录
- 步骤：进入证据校验；执行单个与批量校验
- 期望：成功与失败态均有明确提示；失败显示错误原因

11. 证据包生成与导出
- 前置：已登录
- 步骤：进入互通→证据包；点击生成；分别导出 JSON 与报告
- 期望：触发下载；文件可打开且非空

12. 互通导出（SISTEMA/PAScal/Siemens SET）
- 前置：已登录
- 步骤：进入互通；填写项目ID；依次点击 导出 SISTEMA CSV、导出 PAScal JSON、导出 Siemens SET JSON
- 期望：三次下载均触发；文件名存在；内容类型正确

13. 互通导入无效反馈
- 前置：已登录
- 步骤：在互通页面导入无效的 SISTEMA CSV 与 PAScal JSON
- 期望：对话提示导入 0 条或错误信息，提示清晰

14. RBAC 角色管理
- 前置：已登录
- 步骤：进入 RBAC；加载角色；创建角色；加载权限；分配与移除角色；检查权限
- 期望：操作反馈成功；检查权限显示允许与拒绝结果

15. RBAC 权限拒绝提示
- 前置：已登录
- 步骤：进入 RBAC；填权限键（如 component:view-sensitive）；点击检查
- 期望：显示拒绝或错误提示；文案清晰

16. 系统配置加载与单项设置
- 前置：已登录
- 步骤：进入系统配置；点击加载配置；填写键（如 appName）；点击读取；填写值并设置
- 期望：读取与设置反馈清晰；值更新或成功提示

17. 系统配置更新/重置/导出/导入
- 前置：已登录
- 步骤：在系统配置页面依次点击 更新配置、重置配置、导出配置、导入
- 期望：更新与重置成功提示；导出触发下载；导入成功或错误有明确提示

18. 性能监控（指标与警告）
- 前置：已登录
- 步骤：进入性能监控；点击加载指标与加载警告
- 期望：显示指标列表（operation/count）与警告摘要（message）

19. 性能报告与重置
- 前置：已登录
- 步骤：性能监控页面点击生成报告；填写操作名 eval；点击重置指标
- 期望：报告生成提示；重置后指标回归初始或提示变更

20. 缓存管理（统计与清空、删键）
- 前置：已登录
- 步骤：进入缓存管理；点击加载统计；点击清空缓存；填写键名并删除键
- 期望：统计加载显示条目数；清空与删键反馈成功；清空后统计为 0

21. 组件版本与附件
- 前置：已登录
- 步骤：进入组件版本→加载；进入组件附件→加载；尝试上传、下载、删除附件
- 期望：列表显示完整；上传/下载/删除成功且下载文件非空

22. 模板列表与渲染
- 前置：已登录
- 步骤：进入报告模板；加载模板；点击渲染
- 期望：渲染成功预览；错误时提示明确

23. 合并报告生成
- 前置：已登录
- 步骤：进入合并报告；点击生成
- 期望：成功提示或失败提示清晰，报告可预览或下载

24. 实时计算（创建/计算/取消）
- 前置：已登录
- 步骤：进入实时计算；点击创建会话；点击执行计算；点击取消计算
- 期望：显示会话与计算结果；取消后状态与提示更新

25. 图纸验证
- 前置：已登录
- 步骤：进入电气图纸验证；点击验证
- 期望：显示验证结果列表与细节

26. 方程简化与 SRECS 分解
- 前置：已登录
- 步骤：进入方程简化或 SRECS；执行分析
- 期望：显示分析结果摘要与指标

27. T1/T10D 参数与建议
- 前置：已登录
- 步骤：进入 T1/T10D；调整参数；查看建议
- 期望：提示合理建议；参数更新成功

28. 通道可视化
- 前置：已登录
- 步骤：进入通道可视化；点击加载数据
- 期望：图表渲染正确；数据与图例一致

29. 下载拒绝（负向）
- 前置：已登录
- 步骤：在合规自检页面点击导出PDF（模拟拒绝场景）
- 期望：下载内容为错误信息或提示；包含 forbidden 或错误码

30. 互通导出 403（负向）
- 前置：已登录
- 步骤：在互通页面点击 导出 SISTEMA CSV（模拟 403）
- 期望：显示禁止提示；不产生有效数据文件

31. RBAC 检查 401（负向）
- 前置：已登录
- 步骤：在 RBAC 中执行检查（模拟 401）
- 期望：显示未授权提示；引导用户登录或刷新

32. 性能报告 500（负向）
- 前置：已登录
- 步骤：在性能监控点击生成报告（模拟 500）
- 期望：显示服务器错误提示；不影响后续加载指标

33. 网络中断（负向）
- 前置：已登录
- 步骤：在性能监控点击加载指标（模拟网络断开）
- 期望：显示网络错误或重试提示；系统保持响应

下载校验建议
- 检查浏览器默认下载目录中文件名与大小，确保非空
- PDF/HTML/JSON/CSV 文件可打开且内容类型正确
- 错误场景下载文件包含可读错误信息与关键字（如 forbidden、error）

报告查看
- 自动化覆盖的测试报告可通过命令查看：npx playwright show-report
- 失败时包含截图、视频与 trace（已在配置启用）
