Safetool 前端用户手册（现有功能）

概览
- 登录：输入用户名与密码后进入系统；本地预览可使用示例账号 user/pass。
- 顶部导航：各功能模块以按钮形式展示，点击切换模块。
- 导出/下载：生成的报告、数据可下载为 PDF/HTML/JSON/CSV 等格式。

合规自检（Compliance）
- 目标：进行 ISO12100 与 ISO13849-1 的合规性自检并生成报告。
- 能做什么：
  - 录入系统基本信息与风险参数（严重度/频度/可避性/降低措施）。
  - 配置所需 PL、架构、DCavg、MTTFd、CCF 分数与验证状态。
  - 执行自检得到“合规摘要/不符合项列表”。
  - 生成并预览报告（HTML）、导出 PDF。
  - 同步项目的“验证矩阵摘要”。

IEC 62061 评估（IEC62061）
- 目标：基于安全功能与子系统进行 IEC 62061 评估与报告生成。
- 能做什么：
  - 定义安全功能（目标 SIL、T1、T10D）。
  - 管理子系统与组件属性（制造商/型号/PFHd/beta），可从组件库选择。
  - 执行评估得到 PFHd 与 AchievedSIL。
  - 预览与导出报告（HTML/PDF）。
  - 同步当前功能的“验证矩阵摘要”。

组件库（Library）
- 目标：维护组件库与进行替代建议查询。
- 能做什么：
  - 加载组件列表；新增/删除组件；导入 JSON；导出 JSON/CSV。
  - 针对组件查询替代建议（支持分数阈值、类别/制造商一致、PFHd/DCavg/MTTFd阈值）。

验证矩阵（Matrix）
- 目标：查看项目的条款要求、证据与结果。
- 能做什么：
  - 输入 `项目ID` 加载矩阵项；查看标准、条款、要求、参考、证据编号、结果、责任人、到期日。

验证清单（Verification）
- 目标：按标准加载项目的验证条目。
- 能做什么：
  - 输入 `项目ID` 与标准（如 ISO13849-2），加载条目列表并查看条目标题与条款引用。

模型器（Modeler）
- 目标：构建/管理系统或安全功能的模型数据（占位模块）。
- 能做什么：
  - 在后续版本中用于编辑系统模型、拓扑与参数；当前为占位导航。

互通（Interop）
- 目标：与外部工具进行数据导入/导出（SISTEMA/PAScal/Siemens SET）。
- 能做什么：
  - 输入 `项目ID`，导出 SISTEMA CSV、PAScal JSON、Siemens SET JSON。
  - 导入 SISTEMA CSV 与 PAScal JSON，显示导入统计或错误反馈。
  - 生成证据包并导出 JSON/HTML 报告。

证据库（Evidence）
- 目标：查看与管理证据清单。
- 能做什么：
  - 加载证据列表，查看证据名称、类型与状态。

证据校验（Evidence Validation）
- 目标：校验单个或批量证据数据与关联链路。
- 能做什么：
  - 单个校验，显示通过/失败原因。
  - 批量校验，显示失败列表与错误摘要。
  - 查看“证据链”关联信息。

SRS（安全需求规格）
- 目标：创建安全需求规格文档并导出。
- 能做什么：
  - 定义系统名称、安全功能、PLr、架构类别与参数（DCavg/MTTFd等）。
  - 维护需求列表（标题/描述/接受准则/条款引用/必需）。
  - 创建 SRS；导出 HTML/PDF；AI 草拟（生成初稿文本）。

图纸关联（Electrical Drawing）
- 目标：进行电气图纸相关验证（占位模块）。
- 能做什么：
  - 执行图纸验证并查看结果列表。

双标准评估（Dual Standard）
- 目标：并行进行多个标准评估（占位模块）。
- 能做什么：
  - 同步执行并显示评估结果摘要。

联动整改（Linked Remediation）
- 目标：生成整改建议与联动操作（占位模块）。
- 能做什么：
  - 基于评估结果生成建议列表。

矩阵增强（Matrix Enhancement）
- 目标：提供条款索引与快速检查能力。
- 能做什么：
  - 执行索引；进行条款检查；查看结果摘要。

通道可视化（Channel Visualization）
- 目标：展示通道数据与图表。
- 能做什么：
  - 加载数据并渲染图表；查看图例与指标。

规则分层（Rule Hierarchy）
- 目标：查看规则的分层结构。
- 能做什么：
  - 加载与对比规则层次与内容。

方程简化（Equation Simplification）
- 目标：进行方程模型的简化分析。
- 能做什么：
  - 计算并展示简化结果与关键项。

SRECS 分解（Srecs Decomposition）
- 目标：分解安全相关事件/控制系统要素。
- 能做什么：
  - 展示分解结果与结构化视图。

批量评估（Batch Evaluation）
- 目标：对多个项目或功能执行批量评估。
- 能做什么：
  - 执行批量任务并汇总结果。

离线部署（Deployment Config）
- 目标：生成离线部署配置与文档。
- 能做什么：
  - 配置生成与文档导出。

性能监控（Performance Monitor）
- 目标：监控系统操作性能并生成报告。
- 能做什么：
  - 加载性能指标与警告列表。
  - 生成性能报告；按操作名重置指标。

缓存管理（Cache Management）
- 目标：查看与管理系统缓存。
- 能做什么：
  - 加载缓存统计；清空缓存；按键删除缓存项。

RBAC 权限管理（Rbac Management）
- 目标：管理角色与权限，并进行权限检查。
- 能做什么：
  - 加载与创建角色；为用户分配或移除角色。
  - 加载用户权限；检查权限键允许/拒绝。

系统配置（System Config）
- 目标：管理系统配置与单项设置。
- 能做什么：
  - 加载/更新/重置/导出/导入配置。
  - 单项设置读取/修改（如 appName）。

组件版本（Component Version）
- 目标：查看组件版本信息。
- 能做什么：
  - 加载版本列表与元数据。

组件附件（Component Attachment）
- 目标：管理组件关联的附件。
- 能做什么：
  - 加载附件列表；上传文件；下载与删除附件。

IEC60204 检查（IEC60204）
- 目标：执行 IEC60204 的过载保护检查。
- 能做什么：
  - 运行检查并查看结果与错误提示。

T1/T10D 参数（T1/T10D）
- 目标：管理时间参数与建议。
- 能做什么：
  - 调整参数，查看优化建议。

实时计算（Realtime）
- 目标：建立计算会话并进行实时计算。
- 能做什么：
  - 创建会话；执行计算；取消计算；查看结果与状态。

本地化格式化（Localization）
- 目标：演示单位/时间/百分比/日期/数字等本地化格式化。
- 能做什么：
  - 加载本地化文案并展示格式化效果。

报告模板（Report Templates）
- 目标：管理报告模板并进行渲染。
- 能做什么：
  - 加载模板列表；渲染模板并预览；错误场景提示。

合并报告（Combined Report）
- 目标：生成多数据源合并报告。
- 能做什么：
  - 生成合并报告并查看提示；失败场景显示错误信息。

证据包（Evidence Package）
- 目标：按项目生成证据包并导出。
- 能做什么：
  - 生成后导出 JSON 与报告（HTML）。

统计报表（Statistics）
- 目标：加载系统统计摘要。
- 能做什么：
  - 加载并展示项目数量、证据数量等统计。

设置（Settings）
- 目标：配置 PLr 映射与查询审计日志。
- 能做什么：
  - 加载/修改 PLr 映射规则并保存。
  - 查询审计日志（支持用户/操作/分页筛选）。

使用建议
- 本地预览可用示例账号（user/pass）；后端不可用时部分功能有演示数据回退以方便体验。
- 下载文件请在浏览器的默认下载目录中查看并校验大小与类型。
- 批量操作或导入/导出前，建议先进行小数据验证以确保格式正确。
