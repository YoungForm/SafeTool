# 前端 UI 与流程方案（模型器/库/校核）

## 1. 导航结构
- 评估：ISO 13849 / IEC 62061 切换页签。
- 建模：图形化 I/L/O 模型器（通道与设备拖拽/属性编辑）。
- 库：组件库查询/导入/编辑。
- 验证：ISO13849-2 与 IEC60204-1 清单执行与结果录入。
- 证据：上传与绑定，状态与到期提醒。
- 报告：预览与生成（HTML/PDF）。

## 2. 关键界面
- 模型器：
  - 画布与层级树；设备选取与通道连接；类别/架构实时推导与提示。
  - 右侧属性面板：参数、证据引用与计算结果。
- 评估页：
  - ISO：Category/MTTFd/DCavg/CCF 输入与提示；串联设备 DCavg 方法切换。
  - IEC：子系统架构选择与组件参数输入；PFHd 实时汇总；T1/T10D 校核。
- 库管理：
  - 表格与筛选；批量导入（JSON/CSV）；详情页展示证书与附件。
- 验证清单：
  - 条款→证据→结果的矩阵式录入；整改项与责任分派；状态流转。
- 报告预览：
  - 双语切换；计算明细与证据链；合规矩阵摘要与签名区域。

## 3. 交互与体验
- 实时计算与错误提示（可定位、可修复建议）。
- 快捷操作：从模型器一键创建验证条目与证据占位。
- 可视化风险热度与合规进度。

## 4. 技术实现建议
- 组件：React + TypeScript，状态管理（例如 Zustand/Redux），图形模型器（Canvas/SVG）。
- 国际化：i18n 与单位本地化；模板多语言。
- 性能：虚拟列表与分块加载；后台任务生成报告。
